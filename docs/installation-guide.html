KyOS Installation Guide

Welcome to the comprehensive installation guide for KyOS (Dragon Arch). This guide will walk you through the entire installation process, from downloading the ISO to your first boot into KyOS.

Before You Begin
Make sure to read our System Requirements and back up any important data before installation.

Step 1: Download KyOS

1Visit the download section and choose your preferred edition:

  • Standard Edition: Perfect for most users, includes Qtile and essential tools
  • Security Edition: Includes BlackArch repository with 2000+ security tools
Tip
Always verify the SHA256 checksum after downloading to ensure file integrity.

Step 2: Create Installation Media

2Create a bootable USB drive using your preferred tool:

Linux

# Using dd (replace /dev/sdX with your USB device)
sudo dd if=kyos-2024.12-x86_64.iso of=/dev/sdX bs=4M status=progress
sudo sync

# Or using Ventoy (recommended)
# Install Ventoy on your USB drive and copy the ISO file

Windows

Use tools like:

  • Rufus (recommended)
  • Ventoy
  • Balena Etcher

macOS

# Find your USB device
diskutil list

# Unmount the USB drive (replace diskN with your device)
diskutil unmountDisk /dev/diskN

# Write the ISO
sudo dd if=kyos-2024.12-x86_64.iso of=/dev/rdiskN bs=4m

# Eject the drive
diskutil eject /dev/diskN

Step 3: Boot from USB

3Configure your system to boot from the USB drive:

  1. Insert the USB drive into your computer
  2. Restart your computer
  3. Enter BIOS/UEFI settings (usually F2, F12, DEL, or ESC during boot)
  4. Set USB drive as the first boot device
  5. Save and exit
UEFI Systems
For dual-boot setups on UEFI systems, make sure Secure Boot is disabled in your BIOS settings.

Step 4: Choose Installation Method

4Once KyOS boots, you'll see the desktop. Open a terminal and run:

kyosinstall

The installer will present three options:

Automated Installation

Perfect for new users or fresh installations:

  • Uses sensible defaults
  • Fastest installation method
  • Suitable for dedicated KyOS systems

Dual Boot Installation

Install KyOS alongside your existing operating system:

  • Requires minimum 30GB unallocated space
  • Preserves existing OS
  • Automatic GRUB configuration

Manual Partitioning

Full control for advanced users:

  • Custom partition layouts
  • Multiple filesystems support
  • LVM and encryption options

Step 5: Configure Installation

5Follow the guided setup process:

System Configuration

  • Timezone: Select your timezone
  • Keyboard Layout: Choose your keyboard layout
  • Locale: Set your system language

User Account

  • Username: Your primary user account
  • Password: Strong password recommended
  • Root Password: Administrative account password
  • Hostname: Computer name on the network

Additional Options

  • BlackArch Tools: Install security testing tools
  • NVIDIA Support: Install NVIDIA drivers if applicable
  • Encryption: Enable disk encryption (recommended)

Step 6: Installation Process

6Review your configuration and start the installation:

  1. Verify all settings in the summary screen
  2. Confirm installation to begin the process
  3. Wait for installation to complete (typically 10-30 minutes)
  4. Follow any on-screen prompts
Installation Time
Installation time varies based on your hardware and selected options. The Security Edition takes longer due to additional packages.

Step 7: First Boot

7After installation completes:

  1. Remove the USB drive when prompted
  2. Restart your computer
  3. Select KyOS from the boot menu (if dual-booting)
  4. Log in with your user credentials

Post-Installation Steps

Congratulations! KyOS is now installed. Here are some recommended next steps:

System Updates

# Update the system
sudo pacman -Syu

# Update dotfiles (if applicable)
cd ~/.config && git pull

Essential Applications

# Install additional software
sudo pacman -S firefox vlc git code

# Enable AUR helper (if not installed)
sudo pacman -S yay
