First Boot

Congratulations on installing KyOS! After your first boot, you'll be greeted by the beautiful Qtile desktop environment with a clean, minimal interface.

Login Process

  1. Select "KyOS" from the boot menu (if dual booting)
  2. Enter your username and password at the login screen
  3. Choose "Qtile" as your session (it should be the default)
  4. Press Enter to log in
First Time Tip

If you see a black screen with just a status bar at the bottom, that's normal! Qtile starts with a clean workspace. Press Super + Return to open a terminal.

Basic Navigation

Qtile is primarily keyboard-driven, making it incredibly fast once you learn the basic shortcuts.

Window Management

  • Focus Windows: Use Super + J/K or Super + Arrow Keys
  • Move Windows: Use Super + Shift + J/K
  • Resize Windows: Use Super + H/L
  • Close Window: Use Super + W

Workspace Navigation

  • Switch Workspace: Super + [1-9]
  • Move Window to Workspace: Super + Shift + [1-9]
  • Next/Previous Workspace: Super + Tab

Package Management

KyOS uses pacman, the powerful Arch Linux package manager, along with access to the AUR (Arch User Repository).

Basic Package Commands

Update System

sudo pacman -Syu

Updates all installed packages to their latest versions.

Install Package

sudo pacman -S package-name

Installs a package from the official repositories.

Search Packages

pacman -Ss search-term

Searches for packages containing the search term.

Remove Package

sudo pacman -R package-name

Removes a package from your system.

Pro Tip

KyOS includes yay for easy AUR access. Try yay -S package-name to install AUR packages!

Basic Customization

One of KyOS's strengths is its customizability. Here are some quick ways to make the system your own.

Qtile Configuration

Qtile is configured using Python, making it incredibly flexible: 

# Edit Qtile configuration
nano ~/.config/qtile/config.py

# Restart Qtile to apply changes  
Super + Ctrl + R

Wallpaper

# Set wallpaper using feh
feh --bg-scale /path/to/your/wallpaper.jpg

Terminal Colors

Customize your terminal appearance:

# Edit alacritty config
nano ~/.config/alacritty/alacritty.yml
