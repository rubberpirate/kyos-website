Keybindings Reference

Complete keyboard shortcuts for KyOS Qtile

Modifier Key: In KyOS, Super (Windows key) is the main modifier. All window management shortcuts use Super as the primary modifier.

Window Management

Keybinding Action Description
Super + q Close Window Close the currently focused window
Super + Shift + q Force Close Force close unresponsive window
Super + f Fullscreen Toggle fullscreen mode for current window
Super + Shift + f Toggle Floating Toggle between tiling and floating mode
Super + m Maximize Layout Switch to maximized layout (one window)
Super + n Reset Layout Reset current layout to default

Window Resizing

Keybinding Action Description
Super + Ctrl + h Shrink Left Make window smaller horizontally (left)
Super + Ctrl + l Grow Right Make window larger horizontally (right)
Super + Ctrl + j Shrink Down Make window smaller vertically (down)
Super + Ctrl + k Grow Up Make window larger vertically (up)

Window Navigation

Keybinding Action Description
Super + h Focus Left Move focus to window on the left
Super + j Focus Down Move focus to window below
Super + k Focus Up Move focus to window above
Super + l Focus Right Move focus to window on the right
Alt + Tab Next Window Cycle to next window in current group
Alt + Shift + Tab Previous Window Cycle to previous window in current group

Window Movement

Keybinding Action Description
Super + Shift + h Move Left Move current window left
Super + Shift + j Move Down Move current window down
Super + Shift + k Move Up Move current window up
Super + Shift + l Move Right Move current window right

Groups & Workspaces

Keybinding Action Description
Super + 1 to Super + 9 Switch to Group Switch to group/workspace 1-9

Super + Shift + Tab Previous Group Switch to previous group
Super + Shift + 1 to Super + Shift + 9 Move to Group Move current window to group 1-9
Super + Ctrl + Shift + h Move to Previous Group Move window to previous group
Super + Ctrl + Shift + l Move to Next Group Move window to next group

Customization

All keybindings can be customized by editing ~/.config/qtile/settings/keys.py. The modular configuration makes it easy to add, modify, or remove shortcuts to fit your workflow.

Important: After modifying keybindings, restart Qtile with Super + Ctrl + r to apply changes.

Example: Adding Custom Keybinding

# In ~/.config/qtile/settings/keys.py
Key([mod], "o", lazy.spawn("obsidian"), desc="Open Obsidian"),
Key([mod, "shift"], "p", lazy.spawn("pavucontrol"), desc="Audio settings"),
